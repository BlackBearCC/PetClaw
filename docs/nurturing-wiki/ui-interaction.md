# 交互入口与 UI 设计

[← 返回首页](index.md)

## 设计原则

右键菜单只放**最高频动作**，复杂功能收进**养成面板**。不在菜单里堆功能。

---

## 右键菜单 (3 项)

```
┌───────────────┐
│  🍖 喂一口     │  ← 42号口粮一键喂食 (含冷却检查)
│  📋 养成面板    │  ← 打开完整面板
│  ⚙️ 设置       │
└───────────────┘
```

- "喂一口"是最高频操作，一键触达
- 冷却中显示剩余时间灰态，不隐藏
- 商城/背包/任务/玩耍/治疗 → 全部通过养成面板

---

## 交互入口汇总

| 操作 | 触发 | 效果 |
|------|------|------|
| 点击 | 左键单击 | mood +3, 简短状态气泡 (1s) |
| 抚摸 | 左键长按 | mood +15, intimacy +5, 抚摸动画 |
| 快捷喂食 | 右键 → 喂一口 | 42号口粮 (冷却中灰态+倒计时) |
| 养成面板 | 右键 → 养成面板 / 双击宠物 | 打开 4-tab 面板 |
| 拖拽 | 拖动宠物 | 移动位置 (已有) |
| 文件投喂 | 拖文件到宠物 | file_drop 交互 (已有) |

## 状态气泡规则

宠物头顶气泡 = **被动提示**，不需用户主动查看。

| 条件 | 气泡内容 | 优先级 |
|------|---------|--------|
| hunger <= 60 | "有点饿了~" | 低 |
| hunger <= 30 | "太饿了...先喂喂我吧" (拒绝聊天) | 高 |
| health < 40 | "不太舒服..." | 中 |
| mood < 30 | "心情不太好..." | 中 |
| 升级时 | "升级啦! Lv.N!" | 即时 |
| 任务完成 | "任务完成! 快来领奖~" | 即时 |

自动消失 (3-5s)，同时只显示最高优先级的一条。
